# Pasos para la instalación 👀 👇

 - Dale click al botón de forkear 


 - Darle click al botón de code

 - Copia la URL

 - Abrir tu terminal y poner el comando de 
  git clone  <url> 

 - Entrar a la carpeta del proyecto y abrirlo en tu IDE
